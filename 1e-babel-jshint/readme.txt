This demo allows for using es6 in .es6 files and for running jshint linting.


Pre-reqs for babel:
1) have a package.json
2) $ npm install babel-loader --save-dev
3) $ npm install


Pre-reqs for jshint
1) have a package.json
2) have a valid .jshintrc
2) $ npm install jshint-loader --save-dev
3) $ npm install


# Then run either
$ webpack
# or
$ webpack-dev-server
