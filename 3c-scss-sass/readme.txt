This demo shows the Sass / SCSS loader in action

All CSS in inlined via a <style> tag from a require() statement in app.js.

To see it in action, see the 2 style tags that are injected into the dom.

Note:
Fonts have been removed from bootstrap.css to keep the demo simpler.

Pre-reqs for CSS & Style & Sass Loaders:
1) have a package.json
2) $ npm install css-loader --save-dev
3) $ npm install style-loader --save-dev
4) $ npm install sass-loader --save-dev
5) $ npm install


# Then run either
$ webpack
# or
$ webpack-dev-server
