This demo shows a React project using es6 and jsx.

To see the output of app.es6, run '$ webpack' and look in the build/js/bundle.js file.

Pre-reqs for babel:
1) have a package.json
2) $ npm install babel-loader --save-dev
3) $ npm install

Pre-reqs for React:
1) have a package.json
2) $ npm install react --save
3) $ npm install



# Then run either
$ webpack
# or
$ webpack-dev-server
